## Instalacao do Projeto:

Entrando no projeto:

```
cd Projects/Mini-iFood-API/
```

Derubar processos na porta da aplicação.

```
sudo service apache2 stop
```
> Fazer a senha

Executando containers docker da aplicação.

```
docker compose up -d
```

Esperar containers carregarem.
